# Contributing to Origami UI

Thank you for your interest in contributing. This project is in early development -- contributions and feedback are welcome.

## How to Contribute

1. Fork the repository and create a feature branch from `master`.
2. Make your changes.
3. Ensure the code compiles cleanly with `-Wall -Wextra -pedantic -std=c11` on GCC and Clang.
4. Open a pull request with a clear description.

## Coding Standards

- ANSI C (C11 standard), zero warnings under strict flags.
- 4-space indentation, `snake_case` naming, `ou_` prefix for public API symbols.
- No dynamic memory allocation, no external dependencies, no platform-specific code in the core library.

## Questions?

Open an issue and we will be happy to help.
