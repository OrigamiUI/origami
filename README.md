# Origami UI

A lightweight, portable, immediate-mode GUI library written in ANSI C.

Origami UI provides 25+ UI controls for building graphical user interfaces in tools, editors, games, and embedded systems. It produces a stream of drawing commands that the host application renders using any graphics backend of its choosing.

> **Note:** This project is under active development and is not yet ready for use.

## Features

- Immediate-mode architecture -- no retained widget tree
- 25+ built-in controls (buttons, inputs, sliders, dialogs, tabs, and more)
- Zero external dependencies in the core library
- No dynamic memory allocation
- Portable ANSI C (C11)
- Flexible row-and-column layout system
- Runtime-configurable styling
